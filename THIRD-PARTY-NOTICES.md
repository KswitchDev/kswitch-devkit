# Third-Party Notices

Status: incomplete until EP-227 dependency and image evidence is generated.

Release validation must produce dependency and image notice evidence before
public distribution. Required coverage includes:

- Python dependencies;
- npm dependencies;
- Go modules;
- devkit source and scripts;
- generated clients and schemas;
- documentation assets;
- examples;
- container base images, OS packages, and binary layers.

No public release may proceed with missing, unknown, or prohibited notices.
