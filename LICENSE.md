# KSwitch SDKs and Developer Edition Licence Map

This repository uses a mixed licence model.

## Apache-2.0 SDK Surface

The language SDK client surfaces are licensed under Apache-2.0 unless a file
states otherwise:

- `python/`
- `typescript/`
- `go/`
- `examples/sdk-apache/`

The Apache-2.0 text is available at `LICENSES/Apache-2.0.txt`.

## KSwitch Developer Edition Surface

The runnable Developer Edition devkit, runtime configuration, manifests,
container-image references, local entitlement overlays, and bundled platform
material are source-available under the KSwitch Developer Edition Licence:

- `devkit/`
- `LICENSES/KSWITCH-DEVELOPER-EDITION-LICENSE.md`

Developer Edition is not open source. It is available for permitted local,
non-commercial development, education, demos, SDK integration, and bounded
non-production evaluation only.

## No Implied Commercial Rights

This repository does not grant production, customer-facing, managed-service,
revenue-generating, internal business-operation, hosted-service, or commercial
platform rights. Commercial use requires a separate written agreement with
KSwitch.

## Marks

KSwitch names, logos, marks, and product names are not licensed except as needed
for nominative identification. See `TRADEMARKS.md`.

## Third-Party Material

Third-party notices and dependency licence policy live in:

- `THIRD-PARTY-NOTICES.md`
